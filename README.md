# HahaMeterGui
Добавил библиотеке [HahaMeter от skrepkaq](https://github.com/skrepkaq/HahaMeter) простенький интерфейс типа 11 винды(библиотека [sv_ttk](https://github.com/rdbende/Sun-Valley-ttk-theme)).<br/>
Также она будет менять тему в зависимости какая стоит в системе(определяется библиотекой [darkdetect](https://github.com/albertosottile/darkdetect)).<br/>
Изображение smile.ico принадлежит её [законным авторам](https://www.vectorstock.com/royalty-free-vector/laughing-person-with-tears-and-pointing-vector-12862186). Если необходимо - удалю.

## Установка
Последнюю версию можно скачать как [exe файл](https://github.com/MatXia/HahaMeterGui/releases/latest)(понадобиться установленный [Visual C++ Redistributable Runtime](https://www.techpowerup.com/download/visual-c-redistributable-runtime-package-all-in-one/)) или клонировать репозиторий и докачать библиотеки:<br/>
```
git clone https://github.com/MatXia/HahaMeterGui.git
cd HahaMeterGui
pip install -r requirements.txt
```
